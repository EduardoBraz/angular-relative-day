# angular-relative-day

The filter displays relative day name (according to the console) e.g. today, tomorrow, yesterday and etc.

##Install

`bower install angular-relative-day`

##Usage

###Conroller

`angular.module('myApp',['RelativeDay'])`

###View

`{{date|relativeDay}}`
